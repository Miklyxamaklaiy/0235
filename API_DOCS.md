# API проекта

## Внутренние endpoint'ы

### Получить список организаций
`GET /api/organizations`

Поддерживает параметры:
- `q` — поиск по названию, адресу, городу и категории
- `city` — точное значение города
- `category` — точное значение категории
- `favoritesOnly` — только избранные записи для авторизованного пользователя

Пример:
`/api/organizations?q=помощь&city=Москва&category=Социальная помощь`

### Получить список городов
`GET /api/organizations/cities`

### Получить список категорий
`GET /api/organizations/categories`

### Избранное пользователя
- `GET /api/favorites`
- `POST /api/favorites/toggle`

## Источник наполнения каталога

Каталог заполняется двумя способами:
1. стартовый набор реальных карточек организаций
2. синхронизация через OpenStreetMap Overpass API

Синхронизация запускается из админ-панели и обновляет карточки организаций по России. В проекте используются категории:
- гуманитарная помощь
- социальная помощь
- медицинская помощь
- юридическая помощь
- помощь
