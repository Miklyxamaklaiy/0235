# Плечо помощи

Веб-приложение на ASP.NET Core MVC для мониторинга организаций, оказывающих поддержку ветеранам СВО и их семьям. В каталоге доступны поиск, фильтрация по городу и категории, карта организаций, личный кабинет, заявки на помощь и административная панель.

## Основные возможности

- регистрация и авторизация пользователей
- восстановление пароля
- каталог организаций с картой
- фильтрация и избранное
- создание и сопровождение заявок
- административная панель
- обновление каталога через встроенный парсер открытых источников

## Технологии

- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- Leaflet + OpenStreetMap

## Запуск

```powershell
cd PlechPomoshchi
.\scriptseset-db.ps1
.\scriptsun.ps1
```

После запуска приложение будет доступно по адресу, который покажет ASP.NET Core в консоли.
